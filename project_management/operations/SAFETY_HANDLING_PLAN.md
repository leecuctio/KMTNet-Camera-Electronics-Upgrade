# KMTNet-CEU Safety and Handling Plan

최종 갱신일: 2026-06-22

## 현장 안전 기준

- 카메라 탈거 및 이동 시 중량물 작업 절차를 사전 확인한다.
- CCD 및 전자부 취급 시 ESD 대응을 적용한다.
- 진공 작업 전 O-ring, flange, feedthrough 상태를 확인한다.
- 클린부스 내 작업자는 장갑, wipe, 청정도 관리 절차를 준수한다.
- 현장 작업 시작 전 당일 safety briefing을 수행한다.

## Daily Safety Checklist

- [ ] 작업공간 접근과 동선 확인
- [ ] 카메라 이동 경로 확보
- [ ] ESD 대응 물품 확인
- [ ] 클린부스 상태 확인
- [ ] 진공/냉각 작업 위험 확인
- [ ] 당일 담당자와 stop-work 조건 확인
