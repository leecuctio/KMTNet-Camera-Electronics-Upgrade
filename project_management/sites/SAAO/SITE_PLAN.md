# SAAO Site Plan

최종 갱신일: 2026-06-30

역할: Final Site  
목표 일정: 2026-12-06 to 2026-12-20 (이동 포함 15일, 현장 서덜랜드 9일)

> SSO/CTIO 교훈·SOP를 반영해 적용하는 최종 사이트. CTIO와 동일하게 온스카이 영상으로 확인하되, 칠레보다 날씨가 불리하므로 **온스카이 시험관측을 3박(12-14~16)으로 확보**한다. 출장 기간 중 주말 휴식 없이 연속 작업하며, 12-09는 다인원이 작업을 분담해 병행한다. 케이프타운 도착 후 서덜랜드로 이동한다.

## 선행 조건

- [ ] CTIO Engineering/Science Acceptance 완료
- [ ] SSO/CTIO 교훈을 SOP에 반영
- [ ] 남은 site 적용에 필요한 spare, cable, tool 확인
- [ ] SAAO 적용 전 software/configuration baseline 고정
- [ ] SAAO 현장 화물 도착, 통관, 작업공간, 네트워크 확인
- [ ] 현지 관측자에게 12-08 냉각기 off 지시 (도착 전 warm-up)
