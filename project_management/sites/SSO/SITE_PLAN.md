# SSO Site Plan

최종 갱신일: 2026-06-30

역할: Prototype Site  
목표 일정: 2026-10-19 to 2026-11-01 (이동 포함 14일, 현장 12일)

> 첫 적용 사이트로, 핵심 작업에 충분한 시간을 배분해 실수를 줄인다. 출장 기간 중 주말 휴식 없이 연속 작업한다. 작업 기간 동안 망원경 밸런스 유지를 위해 더미 HE박스를 장착하며, 10-30에 실제 HE박스로 교체한다. SSO acceptance는 기능시험 기반이며, 온스카이 commissioning은 2027-02에 별도 수행한다(올해 목표 외).

## 선행 조건

- [ ] Production Wallboard 납품 및 acceptance 완료
- [ ] Software Demonstration 통과
- [ ] Full Rehearsal 통과
- [ ] 핵심 장비 현지 도착 확인
- [ ] Recovery/Rollback plan 준비
- [ ] Science Verification 사전 검토 완료 (김재우)
- [ ] 화물 도착, 통관, 작업공간, 클린부스, 네트워크, 전원 확인
- [ ] 더미 HE박스 준비

- 시험관측은 최대 10-29까지 수행하고, 그 결과에 따라 10-29에 GO/NOGO를 결정한다.

- 10-30 "신규 HE박스 망원경 설치"는 작업 중 장착했던 더미 HE박스를 떼고 실제 HE박스를 망원경에 설치하는 작업이다.
