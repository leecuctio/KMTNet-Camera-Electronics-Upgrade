# CTIO Site Plan

최종 갱신일: 2026-06-30

역할: Second Site  
목표 일정: 2026-11-14 to 2026-11-26 (이동 포함 13일, 현장 7일)

> SSO 교훈·SOP를 반영해 적용한다. CTIO에서 **신규 전자부의 첫 온스카이 영상**을 확인한다 (SSO는 기능시험으로 acceptance, 온스카이는 2027-02 예정). 출장 기간 중 주말 휴식 없이 연속 작업하며, 도착일(11-16)은 인원이 많아 클린부스 제작·카메라 제거·HE박스 제거를 분담해 병행한다.

## 선행 조건

- [ ] SSO 현장 시험관측/GO-NOGO 결과 및 교훈 검토 완료
- [ ] SSO 교훈을 SOP에 반영
- [ ] SSO에서 확인된 누락 품목, spare, cable, tool 보완
- [ ] CTIO 적용 전 software/configuration baseline 고정
- [ ] CTIO 현장 화물 도착, 통관, 작업공간, 네트워크 확인
- [ ] 현지 관측자에게 11-15 냉각기 off 사전 요청 (도착 전 warm-up)

- CTIO에서 신규 전자부의 **첫 온스카이 영상**을 확보한다. 칠레는 3개 사이트 중 날씨가 가장 좋아 온스카이 관측 가능성이 높다.
- 11-22 온스카이 관측 후 김재우에게 영상 검토를 요청한다. **영상이 정상이면 11-23 예정대로 출발**, **비정상이면 출장 일정을 연장하여 문제를 해결**한다.
